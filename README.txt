--------------------------Requirements----------------------------
needs node.js installed
--------------------------Settings--------------------------------
safeMode: Disables adding the file types exe, cue, iso and img.

--------------------------Compiling-------------------------------
To Compile put files in the folder "Files" Then Open compiler.js -
with node.JS.
then a file will apear when done named file.gcompiled.
Thats it!

--------------------------Decompiling-----------------------------
Make sure folder decompiled is empty. (It can also do it by itself.)
Find the file to decompile then name it file.gcompiled then run -
node.JS. Wait until done, then go into the folder "decoumpiled".
